# Divisiveis por 3 (1 a 100)

Este exercicio imprime todos os numeros entre 1 e 100 que sao divisiveis por 3.

## Objetivo

- Usar um loop `for` para percorrer o intervalo de 1 a 100.
- Usar o operador de modulo (`%`) para verificar divisibilidade por 3.

## Como executar

No terminal, dentro da pasta `divisiveis3`:

```bash
go run divisiveis3.go
```

Opcionalmente, voce pode executar o pacote inteiro:

```bash
go run .
```

## Saida esperada (exemplo)

Os valores impressos serao os multiplos de 3 no intervalo:

```text
3
6
9
...
99
```

## Estrutura

- `divisiveis3.go`: codigo principal do exercicio.
